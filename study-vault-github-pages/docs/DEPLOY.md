# GitHub Pages 배포 방법

1. GitHub에서 새 Repository를 만듭니다.
2. 이 ZIP 파일을 풀고 안의 파일들을 Repository 최상위에 업로드합니다.
3. Repository의 `Settings > Pages`로 이동합니다.
4. `Build and deployment`에서 Source를 `Deploy from a branch`로 선택합니다.
5. Branch는 `main`, Folder는 `/root`로 설정합니다.
6. 저장 후 몇 분 뒤 Pages 주소가 생성됩니다.

## 주의

`index.html`이 Repository 최상위에 있어야 첫 화면이 바로 열립니다.
